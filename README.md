## VExim DNS Core

[![Latest Version on Packagist](https://img.shields.io/packagist/v/mrsleeps/vexim-web-plugin-dns-core.svg?style=flat-square)](https://packagist.org/packages/mrsleeps/vexim-web-plugin-dns-core)

This is core package for the DNS Providers in [VExim Web UI](https://github.com/MrSleeps/VExim-Web-UI).

This gets installed automatically by the package [VExim PDNS](https://github.com/MrSleeps/vexim-pdns)
